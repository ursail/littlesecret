Little Secret Admin

Diese Admin-Seite verwendet Sveltia CMS mit GitHub Access Token. Dadurch brauchst du keinen Netlify-Login.

Wichtig: Nicht auf Sign in with GitHub klicken, sonst wird wieder Netlify-OAuth geöffnet. Verwende Sign in with Token.

Einmalig: GitHub Token erstellen

  1. Öffne GitHub: Fine-grained personal access tokens.
  2. Generate new token wählen.
  3. Repository auswählen: ursail/littlesecret.
  4. Permissions setzen: Contents: Read and write.
  5. Token erstellen, kopieren und im CMS bei Sign in with Token einfügen.

Der Token wird nur in deinem Browser gespeichert. Teile ihn nicht weiter.
